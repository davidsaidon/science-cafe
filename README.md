"""
🧪 SCIENCE CAFÉ — DAVID'S CHEAT SHEET
======================================
10 minutes per group | Ages 5–11 | Laptop ready

BEFORE YOU START:
- Have Claude open in a browser tab (claude.ai)
- Have magic_spiral.py ready to run in a terminal
- Have name_explosion.py ready in another terminal tab
"""

# ─────────────────────────────────────────────
# 🕐 SUGGESTED 10-MINUTE RUN OF SHOW
# ─────────────────────────────────────────────
#
# [0:00 – 1:30]  INTRO
#   "Hi! I'm Noga's dad. I work with computers —
#    but not the boring kind. I help teach computers
#    how to THINK. Has anyone talked to Siri or Alexa?
#    I help build something like that."
#
# [1:30 – 2:30]  SHOW THE CODE (quick wow moment)
#   "This is what I type all day — it looks like
#    a secret language, right?"
#   → Run magic_spiral.py
#   "These few lines of weird text made THAT.
#    That's what code does — you write instructions,
#    and the computer follows them."
#
# [2:30 – 3:30]  OPTIONAL: NAME EXPLOSION
#   → Ask one kid their name, run name_explosion.py
#   "See? The computer can play with your name —
#    spell it backwards, turn it into secret numbers."
#
# [3:30 – 7:00]  THE MAIN EVENT — LIVE AI DEMO
#   → Open Claude in the browser
#   "Now here's the really cool part. This computer
#    can TALK. You can ask it anything."
#
#   Use these prompts to get started:
#
#   🎯 FUN STARTERS (pick 2-3):
#     • "Write a funny 4-line poem about [kid's name]"
#     • "If a dog could code, what would it build?"
#     • "Explain what a black hole is to a 6-year-old"
#     • "What would happen if gravity turned off for 5 minutes?"
#     • "Invent a new animal that's half [X] half [Y]"
#       (let kids pick the animals)
#     • "Write a super short story about a robot who
#        goes to [their school name]"
#
#   🧠 THE "CAN IT GUESS?" GAME:
#     → "I'm thinking of an animal. I'll give you clues
#        one at a time. Try to guess."
#     → Let a kid give clues: "It's big", "It lives in water"
#     → Type each clue, watch Claude narrow down
#
# [7:00 – 9:00]  QUESTIONS FROM KIDS
#   Common questions & quick answers:
#
#   "Is the computer alive?"
#   → "No — it's really good at PRETENDING. It's like
#      how a puppet isn't alive, but a puppeteer makes
#      it seem alive. I'm one of the puppeteers."
#
#   "Can it do my homework?"
#   → "It could try! But it sometimes gets things wrong.
#      Your brain is actually better at lots of things."
#
#   "Will robots take over the world?"
#   → "Computers only do what people tell them to do.
#      They can't want things. They're more like a very
#      clever calculator than a brain."
#
# [9:00 – 10:00]  WRAP UP
#   "The coolest thing about my job is that anyone
#    can learn to do it. If you can give instructions
#    to a friend — like 'go left, go right, pick that up' —
#    you can learn to code. That's all it is:
#    giving a computer very clear instructions."
#
# ─────────────────────────────────────────────
# 💡 TIPS
# ─────────────────────────────────────────────
#
# • Younger kids (5-7): Focus on the spiral + AI chat.
#   Keep it magic, don't over-explain.
#
# • Older kids (8-11): Show the actual code briefly.
#   They'll want to know HOW it works.
#
# • If Wi-Fi is dodgy: The spiral and name scripts
#   work fully offline. Lead with those.
#
# • Energy trick: Let THEM type on the laptop if
#   possible. Ownership = engagement.
#
# • If a kid asks something you don't know:
#   "Great question — let's ask the computer!"
#   (This is genuinely the best move.)
